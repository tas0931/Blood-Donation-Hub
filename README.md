

**Blood Donation Hub:**


Blood Donation Hub is a web application built with the MERN stack (MongoDB, Express.js, React, Node.js) to facilitate blood donation management. It allows users to register as donors or recipients, log in to access role-based dashboards, and enables admins to manage users via an admin dashboard. The application promotes blood donation with a user-friendly interface and robust backend.

**Features (12):**


1.User Registration with Donor and Recipient Based
2.Total user showcase (with total number of donor and recipient separately)
3.Donor and Recipient Based login with Separate dashboard
4.Admin Login panel with unique admin dashboard (One admin Implemented from Dashboard)
5.Admin Dashboard manages users with search (based on blood groups) as well as deletion. (Review of users by Admin)
6.Blood camp page where it shows all recent blood camps with all details in a table (sorted by recent date)
7.Blood camp page has location-based search option.
8.new Blood camp registration with details will be done from admin dashboard by an admin.
9.Blood camp review and deletion option in admin dashboard.
10.Donor and Recipient Dashboard will show case all the information of the logged users
11.Donor Dashboard has Blood group-based Recipient search box and after searching it will show the necessary information of a recipient
12.Recipient Dashboard has Blood group-based Donor search box and after searching it will show the necessary information of a Donor.


Secure Backend: Password hashing with bcrypt and JWT for authentication.

**Technologies Used:**





Frontend: React, Material-UI, Axios, React Router DOM



Backend: Node.js, Express.js, MongoDB, Mongoose



Authentication: JSON Web Tokens (JWT), bcrypt



Environment: dotenv for environment variables



Deployment: https://blood-donation-hub.onrender.com/
